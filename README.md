# KETHAJA — React + Firebase

## Setup Instructions

### 1. Install dependencies
```
npm install
```

### 2. Run locally
```
npm start
```

### 3. Create admin account
Go to Firebase Console → Authentication → Add user
- Email: your email
- Password: your chosen password

### 4. Deploy to Firebase
```
npm install -g firebase-tools
firebase login
npm run build
firebase deploy
```

Your site will be live at: https://kethaja-7f187.web.app

### Admin Panel
Visit: https://kethaja-7f187.web.app/admin
